# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, be constructive, assume good faith. Harassment,
personal attacks, and bad-faith disruption are not tolerated.

Violations can be reported privately to the maintainer via
[GitHub](https://github.com/zubairahmad054) — reports are handled
confidentially.
